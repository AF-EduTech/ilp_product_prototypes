## Kontext
In der Content Delivery gibt es viele interaktive Elemente. Diese werden in der Content Factory generiert und in der Content Delivery ausgspielt. Alle interaktiven Elemente beziehen sich in Unterkapiteln auf Lerninhalte, die vorher behandelt werden.

## Interaktive Elemente
Standard Quizze (werden in allen Kurskategorien verwendet):
- Wahr/Falsch Quiz
- Multiple Choice
- Lückentext
- Freitext
- Zuordnungsquiz

Code-Quizze (werden nur in Programmierkursen verwendet):
- Code Debug (vorgegebener Code mit Fehlermeldung und Musterlösung. Lernender muss anhand Fehlermeldung den Code korrigieren. Lösung muss gegen Musterlösung validiert werden)
- Code-Explanation (vorgegebener Code und Musterlösung. Lernender muss den Code erklären. Lösung muss gegen Musterlösung validiert werden)
- Code Assert (vorgegebener Code und Assertions. Lernender muss den Code so anpassen, dass die Assertions erfüllt sind. Lösung muss gegen Musterlösung validiert werden)

Arbeitsblatt: Einleitung mit einem optionalen Szenario. Mehrere Aufgaben mit Aufgabenstellung und Eingabefeld zur Bearbeitung. Optionale Zusatzaufgaben für den Lernenden.

Selbstreflexion: Einleitung, dann 3-5 Fragen (Stell dir vor..., wie würdest du..., Was denkst du...) mit Eingabefeld zur Beantwortung. 

Lernkarten: Mehrere Lernkarten mit Begriff oder Formulierung auf der Vorderseite und Erläuterung auf der Rückseite.
