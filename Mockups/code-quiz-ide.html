Code-Quiz IDE – Konzept

Integrierte Lernumgebung mit Syntax-Highlighting, Fehlermarkierung und Quiz-Modi

Designkonzept
Integrierte Browser-IDE für Code-Quizze
Die IDE basiert auf einem geteilten Editor-Layout (Split-Pane): links vorgegebener Code mit Syntax-Highlighting und Fehlermarkierungen, rechts der editierbare Bereich des Lernenden. Eine Konsole unterhalb gibt Echtzeit-Feedback zu Syntaxfehlern, Testergebnissen und Hinweisen. Unterstützte Sprachen: PHP, HTML, JavaScript, Java, Python.
